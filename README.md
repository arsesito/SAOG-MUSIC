# SAOG MUSIC

SAOG MUSIC es una innovadora plataforma de música impulsada por inteligencia artificial (IA). Su objetivo es revolucionar la creación musical, permitiendo que la IA desarrolle tanto las letras como las melodías de las canciones, ofreciendo resultados únicos y atractivos.

## Características principales

- **Voz como protagonista**: La IA genera una interpretación vocal con la mayor calidad.
- **Generación de letras**: Creación automática de letras de canciones, adaptadas a diversos géneros musicales.
- **Creación de melodías**: Desarrollo musical creativo con duraciones que van desde los 3 hasta los 5 minutos.
- **Variedad de géneros y estilos vocales**: Soporte para diferentes géneros musicales y estilos de voz, permitiendo personalizar cada creación.
